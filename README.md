## 怎么部署?

记得使用 `python >= 3.8` 版本

### 1. 安装依赖

```bash
pip uninstall websocket
pip install rel requests websocket-client
```

### 2. 部署 [gocqhttp](https://docs.go-cqhttp.org/guide/#go-cqhttp)

- http 端口: `5700`
- 正向 websocket 端口: `5701`

### 3. 运行机器人

```bash
python main.py
```
